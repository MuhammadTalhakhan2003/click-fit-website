# Click Fit - Premium Fitness Website

A professional, responsive fitness website with dark purple and gold theme, animations, API integration, and drag-and-drop image upload functionality.

## Features

- ✅ Professional dark purple & gold premium design
- ✅ Fully responsive UI (mobile, tablet, desktop)
- ✅ Smooth CSS animations and transitions
- ✅ jQuery AJAX API integration (Numbers API)
- ✅ Drag & drop image upload with preview
- ✅ Node.js backend with multer for file handling
- ✅ MySQL database schema with stored procedures
- ✅ Error handling for non-main pages
- ✅ Bootstrap 5 for responsive components
- ✅ Font Awesome icons for enhanced UI
